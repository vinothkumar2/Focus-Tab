# Privacy Policy – Focus Tab

Focus Tab does not collect, store, or transmit any personal or sensitive user data.

## Local Data Storage

All settings, including blocked websites, focus mode state, and timer information, are stored locally within the user’s browser using Chrome’s storage APIs. This data never leaves the user’s device and is not shared with any third parties.

## No Tracking or Analytics

Focus Tab does not use analytics, tracking technologies, advertisements, or external servers. The extension operates entirely offline within the browser environment.

## Contact

If you have any questions about this privacy policy, you can contact the developer via the GitHub repository.
